This is an automation project for [Organo](https://github.com/Cledersonbc/organo).

# Install Node dependecies
```shell
npm install
```

# Run Cypress project

## GUI execution
```shell
npx cypress open
```

## CLI execution
```shell
npx cypress run tests
```

# Requirements
* [Node](https://nodejs.org/en/download/)
* [Git](https://git-scm.com/downloads)
